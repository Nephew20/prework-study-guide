HTML

  • HTML creates the structure of a webpage
  • Uses elements such as text, image, header and list to add content to a page
  • Also utilizes head and body elements
  • Head: contiains info or metadata that helps the browser to render the page correctly (NOT VISIBLE TO THE USER)
  • Body: All elements that we want the user to see and interact with (VISIBLE TO THE USER)

Git

  • Git is a version control system that lets you manage and keep track of versions of your project
  • Git commands start with git
  • Branching allows you to work on the code separate from the main branch so that multiple people can code and make changes at as they see fit
  • To check which branch your on: git status
  • To create a feature branch: git checkout [branch name] i.e. (-b feature/starter-code)
  • To add code to the current working branch: git add, for all changes git add -A
  • To commit the changes from the branch: git commit -m "(descriptive message)"
