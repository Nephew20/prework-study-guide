HTML

  • HTML creates the structure of a webpage
  • Uses elements such as text, image, header and list to add content to a page
  • Also utilizes head and body elements
  • Head: contiains info or metadata that helps the browser to render the page correctly (NOT VISIBLE TO THE USER)
  • Body: All elements that we want the user to see and interact with (VISIBLE TO THE USER)

CSS

  • Cascading Style Sheets (CSS) is the language used for styling your webpage
  • There are three ways to use CSS to style the webpage:
  • Inline CSS: Are coded in line with your code; Allows your to apply style to one singular element
  • Internal Stlye Sheet: Allows you to apply some design for a particular "set" of code
  • External Style Sheet: Most flexible; Allows you to specify CSS rules on a seperate file to make design changes easier. Must link the file
  • To add styles to some and not all elements a class attribute must be used. This allows you to share a CSS rule to any element by making an attribute the selector

Git

  • Git is a version control system that lets you manage and keep track of versions of your project
  • Git commands start with git
  • Branching allows you to work on the code separate from the main branch so that multiple people can code and make changes at as they see fit
  • To check which branch your on: git status
  • To create a feature branch: git checkout [branch name] i.e. (-b feature/starter-code)
  • To add code to the current working branch: git add, for all changes git add -A
  • To commit the changes from the branch: git commit -m "(descriptive message)"

JavaScript

  • JavaScript is a critical tool that allows the user to have interactivity with your website
  • To define a variable in JavaScript you have to use var i.e. var = topic
  • An array is good to use when you are storing a group of related data
  • To place a comment inside of your code use /* (Beginning) */ (End)
